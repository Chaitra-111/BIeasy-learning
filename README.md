# BIeasy Educational Platform

**BIeasy** (pronounced "Buy Easy") is a modern, high-quality online education platform designed for students who want to learn core academic subjects and teachers looking for part-time or full-time teaching opportunities.

## Project Structure

```text
bieasy/
├── assets/
│   └── images/          # Image assets and illustrations
├── index.html           # Home Page
├── subjects.html        # Courses and Subjects listing
├── packages.html        # Pricing and Subject Bundles
├── student-portal.html  # Student Dashboard
├── teacher-portal.html  # Teaching Opportunities & Application
├── certificate.html     # Course Completion Certificate
├── contact.html         # Query & Contact Form
├── styles.css           # Global Styles & Design System
└── app.js               # Main Interaction Logic
```

## Features

### 1. Multi-Page Experience
Built as a proper multi-page website using clean HTML5, CSS3, and Vanilla JavaScript. No heavy frameworks or build tools required.

### 2. Design System
- **Theme:** Educational Blue (`#1e40af`) with a clean, trustworthy aesthetic.
- **Typography:** Modern stack using 'Outfit' for headings and 'Inter' for body text.
- **Components:** Reusable cards, professional forms, and a responsive navigation system.

### 3. Student Features
- **Subject Bundles:** Specialized MPC and BiPC packages with clear pricing and benefits.
- **Enrollment Flow:** Complete enrollment system with package selection and account creation.
- **Authentication:** Secure local login and registration system.
- **Student Portal:** A personal dynamic dashboard displaying student name, active courses, and upcoming live classes.
- **Professional Certification:** Automated certificate generation for course completion.

### 4. Teacher Features
- **Portal:** Dedicated space for teachers to explore full-time and part-time roles.
- **Application:** Clean, validated application form for aspiring mentors.

### 5. Technical Excellence
- **Email Integration:** Forms connected via FormSubmit.co, sending directly to csarts3537@gmail.com.
- **Mobile-First:** Fully responsive across all devices (Desktop, Tablet, Mobile).
- **Sticky Header:** Smooth navigation with glassmorphism effects.
- **SEO Ready:** Semantic HTML structure for better search visibility.

## Setup Instructions

1.  Open the `bieasy` folder.
2.  Open `index.html` in any modern web browser to view the platform.
3.  No installation or server setup required (static site compatible).
